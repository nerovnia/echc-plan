# echc-plan

## Installation

```bash
$ npm install

## After install preparation

Create files:
.env.production
.env.development

Parameters:
PORT=3000
STORAGE='./' + pathToDatabase

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
